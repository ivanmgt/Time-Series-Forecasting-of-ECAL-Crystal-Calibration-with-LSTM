The data used in this work are publicly available on the [Zenodo](https://zenodo.org/records/7510572) platform. The dataset consists of 12 .tar files, each compressed using Gzip, with a total size of approximately 20 GB. This dataset contains experimental measurements from a subset of crystals in the ECAL barrel, recorded between 2016 and 2018 during Run 2 of the LHC. This phase corresponds to the second operational period of the accelerator, spanning from 2015 to 2018, during which the LHC operated at a center-of-mass energy of 13 TeV and achieved the highest collision rate reached up to that time.

The crystal set includes 60 rings from the negative hemisphere within the pseudorapidity range −1.478205 ≤ η ≤ −0.456529, with an average of approximately 360 crystals per ring, and 59 rings from the positive hemisphere within the range 0.0113557 ≤ η ≤ 1.02672, also with an average of approximately 360 crystals per ring, as shown in Figure 16.

Therefore, the dataset does not contain all crystals belonging to the complete ECAL barrel and does not include data from the endcaps. Each .tar file contains 9 or 10 additional files, each corresponding to the data of a different ring, totaling approximately 42,840 crystals.

The data for each ring are stored in .csv format and contain, on average, approximately 17,000 records per crystal.

For more information regarding the metadata associated with the files, visit the repository [FAIR ECAL Monitoring repository](https://github.com/FAIR-UMN/fair_ecal_monitoring/tree/master/metadata?utm_source=chatgpt.com)
